# 🧪 QA Portfolio - SauceDemo

Bem-vindo ao meu portfólio de Qualidade de Software!

Este repositório documenta minha evolução prática como profissional de QA, reunindo artefatos produzidos durante minha jornada de aprendizado e simulações de cenários reais encontrados no mercado.

O projeto utiliza a aplicação **SauceDemo** como sistema de estudo para desenvolver habilidades em testes manuais, testes de API e automação de testes.

## 🎯 Objetivo

Construir um portfólio sólido que demonstre conhecimentos em:

* Testes Manuais
* Planejamento de Testes
* Casos de Teste
* Cenários de Teste
* Reporte de Bugs
* Testes Exploratórios
* Testes Funcionais
* Testes Baseados em Risco
* Testes de API
* Automação de Testes com Cypress
* Git e GitHub
* Boas práticas de QA

## 🛠️ Tecnologias e Ferramentas

* Cypress
* Postman
* Git
* GitHub
* Markdown
* SQL *(em andamento)*
* JavaScript *(em andamento)*
